# Prism Insights - 3 Minute Demo Video Script

## Video Overview
- **Duration:** 3 minutes (180 seconds)
- **Format:** Screen recording with voiceover
- **Resolution:** 1920x1080 (Full HD)
- **Style:** Professional, fast-paced, focused on value

They spend hours on manual analysis instead of growing their business."

- Keep it visual and impactful

**On-Screen Text:**

## RECORDING CHECKLIST

### Before Recording

**Environment Setup:**
- [ ] Clean desktop (no personal files)
- [ ] Close all unnecessary applications
- [ ] Disable notifications (Windows: Focus Assist, Mac: Do Not Disturb)
- [ ] Set screen resolution to 1920x1080
- [ ] Test microphone and audio levels
- [ ] Prepare glass of water nearby

**Application Setup:**
- [ ] Start backend: `cd data-simulator && python main.py`
- [ ] Start frontend: `npm run dev`
- [ ] Verify data is loading: http://localhost:8000/health
- [ ] Open browser to http://localhost:3000
- [ ] Clear browser cache and cookies
- [ ] Set browser zoom to 100%
- [ ] Close extra browser tabs

**Recording Software:**
- [ ] OBS Studio configured (1920x1080, 30fps)
- [ ] Microphone selected and tested
- [ ] Audio levels checked (speak at normal volume)
- [ ] Recording destination folder set
- [ ] Cursor highlighting enabled (optional)

### During Recording

**Speaking Tips:**
- Speak clearly and at moderate pace
- Pause briefly between sections
- Emphasize key numbers and benefits
- Sound enthusiastic but professional
- Don't rush - 3 minutes is enough time

**Navigation Tips:**
- Move cursor smoothly and deliberately
- Pause on important information
- Type queries slowly enough to read
- Let AI responses fully appear
- Highlight key numbers by hovering

**If You Make a Mistake:**
- Don't stop recording
- Pause for 3 seconds
- Restart from beginning of that section
- Edit out mistakes later

### After Recording

**Editing Checklist:**
- [ ] Trim beginning and end
- [ ] Remove mistakes and long pauses
- [ ] Add fade in/out transitions
- [ ] Add background music (subtle, royalty-free)
- [ ] Add on-screen text overlays
- [ ] Add zoom effects on key numbers
- [ ] Verify audio levels are consistent
- [ ] Export as MP4, 1080p, H.264

**Final Review:**
- [ ] Watch entire video
- [ ] Check audio quality
- [ ] Verify all text is readable
- [ ] Confirm timing is under 3 minutes
- [ ] Test on different devices

## TIPS FOR SUCCESS

### Audio Quality
- Record in quiet room
- Use external microphone if possible
- Speak 6-12 inches from microphone
- Do audio test before full recording
- Use noise reduction in post-production

### Visual Quality
- 1920x1080 resolution minimum
- 30fps frame rate
- Clean, uncluttered screen
- Smooth cursor movements
- Proper lighting if showing face

### Pacing
- Don't rush - 3 minutes is plenty
- Pause between sections for editing
- Let important information stay on screen
- Match voiceover to visuals

### Engagement
- Start with the problem (hook viewers)
- Show real functionality (not mockups)
- Highlight specific numbers (credibility)
- End with clear call to action
